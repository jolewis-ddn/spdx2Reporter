# spdx2Reporter

Generate a simple license report from SPDX2 RDF file

## Author

jolewis-ddn

## Installation

 1. `npm -i`

## Usage

 1. Generate SPDX2 RDF file (e.g. from fossology)
 1. `node app.js filename.rdf licensePrefixToTrim`

## Dependencies

- debug
- sax
